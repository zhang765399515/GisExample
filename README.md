# GIS学院

```bash
# 克隆项目
git clone http://git.igeokey.com/qhs/vite-pgEarth.git

# 进入项目目录
cd vite-pgEarth

# 安装依赖
npm install

# 如安装依赖失败请下载
# geokey-gis
npm install git+http://git.igeokey.com/dhy/geokey-gis.git

# vite-plugin-geokey-gis GisDemo + VITE 项目构建插件
npm install git+http://git.igeokey.com/dhy/vite-plugin-geokey-gis.git

# 启动服务
npm run dev

# 构建生产环境 npm run build
# 前端访问地址 http://localhost:8080

## 技术栈
vue3 + vite + GisDemo

node版本建议使用18+
```
